Pulse

2022

PULSE is a catalogue documenting the relationship between biological responses and typography. The catalogue includes over 60 images representing the changes in my heartbeat as I interact with letters in a virtual space.

I built a Unity environment in which I can navigate along the strokes of each letter of the alphabet using a mouse. Connected to a PPG (photoplethysmogram) sensor, I recorded my pulse line as I traced along the letters. The software creates a model of my trails, where the height of each point indicates my pulse during interaction.

The 3D models of each letter were exported, edited, and rendered, constituting a “typeface”. The compositions included in the catalogue using the typeface, perhaps, add a glimpse into how we interact with typography in a virtual space.
